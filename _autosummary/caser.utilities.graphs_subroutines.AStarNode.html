caser.utilities.graphs_subroutines.AStarNode

class AStarNode(parent=None, position=None)[source]

Bases: object

A node class for A* Pathfinding parent is parent of the current Node position is current position of the Node in the maze g is cost from start to current Node h is heuristic based estimated cost for current Node to end Node f is total cost of present node i.e. : f = g + h x and y are Cartesian coordinates of the node
